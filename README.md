# UNO_V2

Ce projet consiste à recréer le **jeu de Uno** en suivant les règles officielles. Il se joue dans la console.

## Objectif

Ce projet est personnel et a pour but de mettre en avant mes premières notions apprises en Java :  

- **Programmation orientée objet** : objets, classes, constructeurs, instances, méthodes  
- **Encapsulation** : public, private, getters et setters  
- **Héritage** : classes mère/fille  
- **Abstraction** : classes et méthodes `abstract`  
- **Interfaces**  
- **Énumérations et collections**

## Installation et lancement

1. Cloner le projet :  
```bash
git clone https://github.com/claire-vaugarny/UNO_V2.git
````
2. Ouvrir le projet dans Eclipse ou tout autre IDE Java.

3. Compiler et lancer la classe Main.java pour démarrer le jeu dans la console.
